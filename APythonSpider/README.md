<h1 align="center">EXNPK论坛关于对百度网盘、微云失效<br>链接模型资源贴统一锁沉处理的通知</h1>

<p align="center">EXNРK发〔2018〕8号</p>

EXNРK全体用户：

　　目前管理组注意到这样的现象，出现不少下载链接失效的模型资源发布帖子仍然被顶起的现象，使得用户费大力气注册账号回复却仍然无法得到资源，这在一定程度上也会给新用户留下不好的印象。目前管理组通过爬虫的方式，逐一排查出所有的微云链接和百度网盘链接失效的帖子。粗略估计一共有248张帖子中的334条链接已经被确定为失效链接。为了减少不必要的麻烦，管理组决定将这部分帖子暂时进行关闭和下沉处理。

　　各位作者如果发现自己有被关闭下沉的帖子，请到本帖里面使用Ctrl+F搜索帖子的链接（http://bbs.exnpk.com/thread-XXXX-1-1.html 这样的格式），查看有无各位发布的模型资源帖子。如果有的话请重新设置分享链接，然后在本帖回复通知链接已经修复，管理组测试链接可用后即重新打开并提升帖子。谢谢各位合作。

　　附件：所有链接已经失效的帖子。

<p align="right">EXNPK管理组</p>
